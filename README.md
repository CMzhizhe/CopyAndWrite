# CopyAndWrite
读取assets下的目录文件，然后写入到android的catch目录

# 目的
平常我们copy一分代码文件到另外的项目里面，我们会发现，这个文件的创建时间是没有改变的，我想修改其文件的创建时间

# 教程
copy你的代码，然后再assets目录下面创建一个文件夹，类似我截图里面的common即可，剩下的就是启动android，即可
